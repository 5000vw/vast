Desktop Installation Guide

The windows Desktop version of our Appllication Requires Windows 10 latest version installed. Confirm this before proceeding with steps.

Download and Open The App Certificate file first. After opening the file , Follow the following steps:

  • 1.

    Click Install Certificate

  • 2.

    Choose Local Machine then click Next

  • 3.

    Choose Place All Certificates in the following Store then click Browse

  • 4.

    Browse and choose Trusted People then OK

  • 5.

    Click Next then click Finnish

  • 6.

    You will get a dialog confirming successful installation of the certificate in your PC

Now open Windows Desktop X64 Apllication and Click install Once installed click Launch

**Important Note** :

This requires Windows 10 installed in your windows pc
